# Kenzie-Test

### Teste feito para a seletiva da Kenzie Academy.

## Instalação

- Para criar um ambiente virtual:

```
virtualenv -p python3 venv
```

- Para ativar o ambiente virtual:
```
source venv/bin/activate 
```

- Para rodar a aplicação:

```
python ./app.py
```

- Url da aplicação:

```
http://localhost:5000/
```
